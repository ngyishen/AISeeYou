# AISeeYou

AISeeYou is a browser extension that scans text on webpages and estimates the probability that the content was AI-generated using fully local ONNX-based machine learning inference running directly inside the browser.

AISeeYou works in real time while browsing social media platforms, analyzing posts, articles, blogs, tweets, and other content without sending data to external servers.

The extension can either highlight suspicious text or block it entirely, functioning similarly to an ad blocker for AI-generated content.

AISeeYou estimates the probability of AI-generated writing using machine learning models. Results are probabilistic and should NEVER be treated as definitive proof of authorship.

AISeeYou is currently tested primarily on Google Chrome. Support for additional browsers and websites may vary.

---

# Features

## Fully Local AI Detection

All inference runs locally inside the browser using:

* ONNX Runtime Web
* WebAssembly (WASM)
* Local ONNX models

No external API or cloud inference is required.

---

## Real-Time Detection

AISeeYou continuously scans visible text while browsing and automatically analyzes qualifying content in real time.

Currently tested platforms include:

* Facebook
* X (Twitter)
* Threads
* LinkedIn
* Reddit

Support quality may vary between platforms due to frequent website changes.

---

## Highlight Mode

Detected content is highlighted directly on the page:

* Red → highly likely AI-generated
* Orange → uncertain / mixed
* Green → likely human-written

Each scanned section displays a confidence badge such as:

```text
AI 97.4%
````

---

## Block Mode

Instead of highlighting content, AISeeYou can block or obscure posts that exceed a configurable AI probability threshold.

---

## Adjustable User Preferences

Users can configure:

* Minimum word threshold
* Detection threshold
* Highlight or block mode

For example, the minimum word threshold can be increased to ignore short comments, UI elements, and small text fragments.

This helps reduce false positives and unnecessary processing.

---

## Export Detection Logs

Detection results can optionally be exported as JSON.

Stored data may include:

* detected text
* word count
* AI probability
* timestamp
* platform source

This feature may be removed or redesigned in future versions due to privacy considerations.

# Installation

## 1. Download the Extension

Clone or download the repository.

---

## 2. Open Chrome Extensions

Open:

```text
chrome://extensions
```

Then:

* Enable "Developer Mode"
* Click "Load unpacked"
* Select the `extension/` folder

# Settings

| Setting       | Description                             |
| ------------- | --------------------------------------- |
| Minimum Words | Ignore text below this word count       |
| Mode          | Highlight or block AI-generated text    |
| Threshold     | AI probability required before blocking |
| Export Logs   | Export detection results as JSON        |

---

# Privacy

AISeeYou performs inference locally inside the browser.

No scanned content is uploaded to external servers.

No cloud-based AI inference is used.

---

# Known Limitations

* Does not work on every website
* Website DOM structures frequently change
* Short posts are intentionally ignored
* AI detection is probabilistic and may produce false positives or false negatives
* Performance may vary depending on browser and hardware
* Some platforms aggressively restrict browser automation behavior

---

# Technical Overview

AISeeYou uses:

* ONNX Runtime Web
* WebAssembly (WASM)
* DistilBERT-based text classification
* Browser-side inference queues
* Mutation observers for real-time scanning

The extension scans only visible content currently loaded in the webpage DOM.

---

# Disclaimer

AISeeYou is intended for research and educational purposes.

Detection results should not be used as definitive evidence of whether content was written by a human or generated by AI.
